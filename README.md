# Fresh Ubuntu Setup

`git clone https://github.com/mkinkela1/setup.git`

`cd setup`

For ubuntu:
`chmod +x install_script.sh`
`./install_script.sh`

For MacOS:
`chmod +x install_script_mac.sh`
`./install_script_mac.sh`
